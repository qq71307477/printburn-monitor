## Key Issues to Address

### 1. 本地验证问题
- **问题**: 无 Qt 环境，只能依赖 GitHub CI
- **方案**: 创建静态检查脚本，无需编译即可发现问题

### 2. 代码依赖复杂
- **问题**: 插件架构、头文件共享、静态成员链接冲突
- **方案**:
  - 头文件独立性检查（确保每个头文件可独立编译）
  - 声明定义匹配检查
  - 静态成员唯一性检查

### 3. 增量开发不一致
- **问题**: 接口与实现不匹配、测试与代码不同步
- **方案**:
  - 全量代码审查
  - 测试文件更新
  - CI 验证

## Decisions Made

| Decision | Rationale |
|----------|-----------|
| 优先静态检查 | 无需编译环境即可发现问题，减少 CI 周期 |
| 保持 Qt 5.12.12 兼容性 | 用户明确要求使用此版本 |
| 分层验证 | 语法层 → 声明层 → 依赖层 → 集成层 |

## 1. 系统概述

### 1.1 系统简介

打印刻录安全监控系统（单机版）是一套符合保密管理规定的涉密打印/刻录安全监控与审计系统。该系统实现打印/刻录任务的全流程管理、审批、审计，支持Windows及国产操作系统（统信UOS/麒麟）。

本系统遵循三员分立原则，确保系统管理的安全性和可追溯性，满足政府机关、军工单位、科研院所、事业单位等对安全打印、光盘刻录的管理需求。

### 1.2 功能特点

- **全流程管控**：从申请、审批到输出执行的完整流程管理
- **三员分立**：系统管理员、安全保密员、安全审计员三权分立
- **密级管理**：支持秘密、机密、绝密等多级密级配置
- **审批流转**：灵活的审批员配置和审批流程
- **条码追溯**：支持二维码/条形码标识，实现输出物追溯
- **日志审计**：完整的操作日志和登录日志记录
- **敏感词检查**：自动检测文件中的敏感内容
- **跨平台支持**：支持Windows、统信UOS、麒麟等操作系统

### 1.3 运行环境

| 平台 | 版本要求 |
|------|----------|
| Windows | Windows 10 1903+ / Windows 11 |
| 统信 UOS | V20+ |
| 麒麟 | V10+ |

**硬件要求**：
- CPU：双核及以上
- 内存：4GB及以上
- 硬盘：500MB可用空间（数据库会随使用增长）

## 2. 快速入门

### 2.1 系统安装

#### Windows系统安装

1. 双击运行安装程序 `PrintAuditSystem-Setup.exe`
2. 按照安装向导提示选择安装路径
3. 等待安装完成
4. 桌面将生成快捷方式图标

#### Linux系统安装

1. 打开终端，进入安装包所在目录
2. 执行安装命令：
   ```bash
   sudo dpkg -i print-audit-system_5.3.0_amd64.deb
   ```
3. 如遇依赖问题，执行：
   ```bash
   sudo apt-get install -f
   ```

系统首次安装后，提供三个三员账号：

| 角色 | 用户名 | 默认密码 |
|------|--------|----------|
| 系统管理员 | admin | admin123 |
| 安全保密员 | security | security123 |
| 安全审计员 | auditor | auditor123 |

> **重要提示**：首次登录后请立即修改默认密码！

## 3. 功能操作

### 3.1 登录与个人主页

#### 登录操作

1. 启动系统后显示登录界面
2. 输入用户名（必填）
3. 输入密码（必填）
4. 可勾选"记住密码"便于下次登录
5. 点击"登录"按钮验证身份
6. 登录失败将显示错误提示，连续失败超过限制将锁定账号

#### 个人主页

登录成功后默认进入个人主页，显示以下内容：

**个人信息区**：
- 用户头像
- 用户姓名
- 密级等级
- 所属部门

**我的任务区**：
以列表形式显示当前用户的任务，包含以下字段：

| 字段 | 说明 |
|------|------|
| 序号 | 任务编号 |
| 文档名称 | 任务涉及的文档名称 |
| 密级 | 文档密级 |
| 类型 | 打印/刻录 |
| 份数 | 输出份数 |
| 申请时间 | 任务提交时间 |
| 状态 | 待审批/已通过/已拒绝/已完成等 |
| 操作 | 查看详情 |

**任务状态说明**：
- 待审批：等待审批员审核
- 已取消：用户主动取消
- 审批通过：审批通过，待执行
- 审批拒绝：审批未通过
- 执行中：正在执行输出
- 执行失败：输出过程出错
- 已完成：成功完成输出
- 已过期：超时未执行

### 3.2 打印申请

#### 操作步骤

1. 点击左侧导航栏"打印申请"
2. 系统自动获取待打印文件信息
3. 填写/选择申请信息

**基本信息**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 文件名 | 只读 | - | 自动显示待打印文件名 |
| 密级 | 下拉选择 | 是 | 从系统配置的密级列表选择 |
| 打印机 | 下拉选择 | 是 | 从已授权的打印机列表选择 |
| 审批员 | 下拉选择 | 否 | 根据密级自动匹配可选审批员 |
| 用途 | 下拉选择 | 否 | 内部/对外/其他 |

**打印格式设置**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 份数 | 数字输入 | 是 | 1-100份 |
| 方式 | 下拉选择 | 是 | 单面/双面(翻转长边)/双面(翻转短边) |
| 颜色 | 下拉选择 | 是 | 黑白/彩色 |
| 纸张 | 下拉选择 | 是 | A4/A3/B5等 |
| 方向 | 下拉选择 | 是 | 纵向/横向 |
| 打印范围 | 下拉选择 | 是 | 全部/自定义页码 |

4. 点击"申请"按钮提交
5. 系统进行敏感词检查
6. 如有敏感词，系统提示警告，用户确认后可继续提交
7. 提交成功后任务进入待审批状态

#### 注意事项

- 确保选择的打印机有使用权限
- 密级选择需与文档实际密级相符
- 文件过大时上传可能需要等待

### 3.3 刻录申请

#### 操作步骤

1. 点击左侧导航栏"刻录申请"
2. 填写申请基本信息

**基本信息**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 任务名称 | 文本输入 | 是 | 长度不超过30字符 |
| 密级 | 下拉选择 | 是 | 从密级列表选择 |
| 份数 | 数字输入 | 是 | 刻录光盘数量 |
| 刻录机 | 下拉选择 | 是 | 从已授权的刻录机列表选择 |
| 刻录类型 | 下拉选择 | 是 | 全部/仅打印盘面/仅刻录数据 |
| 审批员 | 下拉选择 | 否 | 根据密级自动匹配 |
| 申请原因 | 下拉选择 | 否 | 会议输出/日常工作/其他 |

**盘面打印选项**（当刻录类型包含打印盘面时）：
- 打印部门名称：勾选后在盘面打印部门信息
- 打印申请人：勾选后在盘面打印申请人姓名
- 单位名称：填写打印在盘面的单位名称

3. 添加刻录文件
   - 点击"添加文件"选择单个文件
   - 点击"添加文件夹"选择整个文件夹

4. 文件列表显示：

| 字段 | 说明 |
|------|------|
| 文件名称 | 文件名 |
| 大小 | 文件大小 |
| 来源 | 文件路径 |
| 密标 | 密级标识 |
| 删除 | 移除文件 |

5. 确认文件总大小未超过光盘容量
6. 点击"申请"按钮提交

#### 注意事项

- 单个光盘容量有限，注意文件总大小
- 刻录前确保刻录机中有空白光盘
- 刻录类型选择后无法更改

### 3.4 打印输出

#### 操作步骤

1. 点击左侧导航栏"打印输出"
2. 列表显示所有审批通过的打印任务

**任务列表字段**：

| 字段 | 说明 |
|------|------|
| 文档名称 | 待打印文档名 |
| 任务密级 | 文档密级 |
| 打印机 | 指定打印机 |
| 页数 | 文档页数 |
| 份数 | 打印份数 |
| 申请时间 | 任务提交时间 |
| 大小 | 文件大小 |
| 来源 | 文件来源路径 |
| 状态 | 任务状态 |

3. 勾选要执行的任务
4. 点击"打印"按钮开始执行
5. 打印过程信息区域实时显示执行日志
6. 进度条显示任务完成进度
7. 点击"刷新"可更新任务列表

#### 执行流程

1. 系统加载加密的任务文件
2. 解密文件内容
3. 根据打印参数设置打印机
4. 添加条码/水印（如已配置）
5. 执行打印输出
6. 记录执行日志
7. 更新任务状态

### 3.5 刻录输出

#### 操作步骤

1. 点击左侧导航栏"刻录输出"
2. 列表显示所有审批通过的刻录任务

**任务列表字段**：

| 字段 | 说明 |
|------|------|
| 文档名称 | 任务名称 |
| 任务密级 | 光盘密级 |
| 刻录机 | 指定刻录机 |
| 光盘类型 | CD/DVD等 |
| 份数 | 刻录份数 |
| 申请时间 | 任务提交时间 |
| 大小 | 文件总大小 |
| 状态 | 任务状态 |

3. 确保刻录机中已放入空白光盘
4. 勾选要执行的任务
5. 点击"刻录"按钮开始执行
6. 刻录过程信息区域实时显示执行日志
7. 进度条显示刻录进度
8. 刻录完成后系统自动验证数据

#### 刻录类型说明

- **全部**：打印盘面 + 刻录数据
- **仅打印盘面**：只在光盘盘面打印标识
- **仅刻录数据**：只刻录数据文件

### 3.6 任务审批

#### 操作步骤

1. 点击左侧导航栏"任务审批"
2. 系统显示待审批任务列表

**筛选条件**：
- 文档名称：模糊搜索
- 状态：待审批/已审批
- 密级：按密级筛选
- 类型：打印/刻录
- 时间：按时间段筛选

3. 查看任务详情
   - 点击"详情"按钮查看完整信息
   - 显示文件基本信息、申请人信息、打印/刻录参数等

4. 执行审批操作
   - 点击"审批"按钮打开审批窗口
   - 选择"同意"或"拒绝"
   - 填写审批意见（可选）
   - 确认提交

#### 批量审批

1. 勾选多个待审批任务
2. 点击"批量审批"按钮
3. 选择审批结果（同意/拒绝）
4. 填写统一的审批意见
5. 确认提交

#### 审批规则

- 审批员密级必须大于等于任务密级
- 审批后任务状态即时更新
- 审批操作记录日志
- 审批拒绝需填写原因

### 3.7 代理审批设置

代理审批功能允许审批员在无法处理审批时，将审批权限临时委托给其他人员。

#### 设置代理审批

1. 点击左侧导航栏"系统管理" -> "代理审批设置"
2. 点击"新增代理"按钮
3. 填写代理信息：

| 字段 | 说明 |
|------|------|
| 原审批人 | 当前登录用户（自动填充） |
| 代理审批人 | 选择代理人员 |
| 生效密级 | 代理审批的密级范围 |
| 任务类型 | 打印/刻录/全部 |
| 生效日期 | 代理开始日期 |
| 失效日期 | 代理结束日期 |
| 是否启用 | 启用/禁用代理 |

4. 点击"保存"确认设置

#### 管理代理审批

- 查看已设置的代理列表
- 编辑已有代理设置
- 删除代理设置
- 启用/禁用代理

## 4. 系统管理

系统管理功能仅对三员账号可见，不同角色有不同的管理权限。

### 4.1 用户管理

**操作入口**：系统管理 -> 用户管理
**操作角色**：系统管理员

#### 新增用户

1. 点击"新增用户"按钮
2. 填写用户信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 用户姓名 | 是 | 用户真实姓名 |
| 登录账号 | 是 | 登录用户名，系统唯一 |
| 卡号 | 否 | 身份卡号 |
| 密码 | 是 | 登录密码 |
| 所属组织 | 是 | 所属部门 |
| 用户角色 | 是 | 可多选 |
| 密级 | 是 | 用户密级等级 |
| 启用状态 | 是 | 启用/停用 |

3. 点击"保存"创建用户

#### 编辑用户

1. 在用户列表中选择用户
2. 点击"编辑"按钮
3. 修改用户信息
4. 点击"保存"确认修改

#### 删除用户

1. 选择要删除的用户
2. 点击"删除"按钮
3. 确认删除操作
4. 已有关联任务的用户无法删除

#### 批量导入

1. 点击"批量导入"按钮
2. 下载导入模板
3. 按模板格式填写用户数据
4. 上传Excel文件
5. 系统验证数据并导入

**导入Excel格式**：

| 用户姓名 | 登录账号 | 卡号 | 密码 | 所属组织 | 用户角色 | 密级 |
|----------|----------|------|------|----------|----------|------|
| 张三 | zhangsan | | 123456 | 技术部 | 普通用户 | 秘密 |

#### 批量导出

1. 选择要导出的用户（不选则导出全部）
2. 点击"批量导出"按钮
3. 选择导出格式（Excel/CSV）
4. 系统生成文件并下载

#### 重置密码

1. 选择用户
2. 点击"重置密码"按钮
3. 输入新密码
4. 确认重置

### 4.2 角色管理

**操作入口**：系统管理 -> 角色管理
**操作角色**：系统管理员

#### 系统内置角色

| 角色名称 | 说明 |
|----------|------|
| 系统管理员 | 负责用户、角色、组织、设备管理 |
| 安全保密员 | 负责密级、审批员、流水号、条码配置 |
| 安全审计员 | 负责日志审计、操作追溯 |
| 任务审批员 | 审批打印/刻录任务 |
| 任务管理员 | 管理任务流程 |
| 普通用户 | 提交打印/刻录申请 |
| 部门管理员 | 管理本部门用户 |
| 回收员 | 回收已输出的载体 |

#### 角色操作

**新增角色**：
1. 点击"新增角色"按钮
2. 输入角色名称
3. 配置角色权限
4. 保存

**编辑角色**：
1. 选择角色
2. 点击"编辑"
3. 修改角色信息和权限
4. 保存

**删除角色**：
- 内置角色不可删除
- 已分配用户的角色需先移除用户关联

### 4.3 组织架构

**操作入口**：系统管理 -> 组织架构
**操作角色**：系统管理员

#### 组织架构操作

**新增部门**：
1. 选择上级部门（顶级部门选择根节点）
2. 点击"新增部门"按钮
3. 输入部门名称
4. 保存

**编辑部门**：
1. 选择部门节点
2. 点击"编辑"按钮
3. 修改部门名称
4. 保存

**删除部门**：
- 存在下级部门的需先删除下级
- 存在关联用户的部门需先移除用户

**部门排序**：
- 通过拖拽调整部门顺序
- 通过上下移动按钮调整

### 4.4 设备管理

**操作入口**：系统管理 -> 设备管理
**操作角色**：系统管理员

#### 设备类型

- 打印机
- 刻录机

#### 新增设备

1. 点击"新增设备"按钮
2. 填写设备信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 设备名称 | 是 | 设备显示名称 |
| 设备编号 | 是 | 设备唯一编码 |
| 设备类型 | 是 | 打印机/刻录机 |
| IP地址 | 否 | 设备网络地址 |
| MAC地址 | 否 | 设备物理地址 |
| 品牌型号 | 否 | 设备品牌和型号 |
| 安装位置 | 否 | 设备物理位置 |
| 关联密级 | 是 | 设备可处理的最高密级 |
| 设备状态 | 是 | 正常/脱机 |

3. 选择提交部门或用户

## 6. 常见问题

### 6.1 登录问题

**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码。三员账号密码遗忘需联系其他管理员协助。

**Q: 登录提示"账号已锁定"？**
A: 连续登录失败超过限制次数后账号会被锁定，请等待锁定时间结束或联系管理员解锁。

**Q: 登录后立即退出？**
A: 可能是会话超时设置过短，或数据库连接异常，请联系管理员检查。

### 6.2 打印问题

**Q: 打印机列表为空？**
A: 可能原因：
1. 未配置打印机设备，联系管理员添加
2. 未获得打印机使用授权，联系管理员授权
3. 打印机状态为脱机，联系管理员启用

**Q: 打印任务提交后无反应？**
A: 检查网络连接，确认系统服务正常，必要时重启应用程序。

**Q: 打印输出失败？**
A: 可能原因：
1. 打印机离线或缺纸
2. 文件损坏或格式不支持
3. 系统权限不足
查看执行日志获取详细错误信息。

### 6.3 刻录问题

**Q: 刻录机无法识别光盘？**
A:
1. 确认刻录机连接正常
2. 更换空白光盘重试
3. 检查刻录机驱动状态

**Q: 刻录失败提示空间不足？**
A: 文件总大小超过光盘容量，请减少文件或更换更大容量光盘。

**Q: 盘面打印模糊？**
A: 检查打印机墨盒状态，清洁打印头。

### 6.4 审批问题

**Q: 看不到待审批任务？**
A:
1. 确认账号具有审批员角色
2. 检查审批员配置是否正确
3. 确认密级匹配

**Q: 无法审批某些任务？**
A: 审批员密级必须大于等于任务密级，低密级审批员无法审批高密级任务。

### 6.5 系统管理问题

**Q: 无法删除用户？**
A: 用户已关联历史任务，为保证审计完整性，需先停用账号而非删除。

**Q: 无法删除密级？**
A: 该密级已被用户或任务引用，需先移除关联。

**Q: 配置修改后未生效？**
A: 部分配置需要刷新页面或重新登录后生效。

### 6.6 其他问题

**Q: 系统响应缓慢？**
A:
1. 检查数据库是否过大，可清理历史日志
2. 检查系统资源占用情况
3. 重启应用程序

**Q: 如何备份数据？**
A: 联系系统管理员进行数据库备份，备份文件需妥善保管。

**Q: 如何迁移到新电脑？**
A:
1. 在原电脑导出用户数据
2. 在新电脑安装系统
3. 导入用户数据和配置
建议由管理员操作
